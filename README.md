# 🎓 Projeto Cartão de Alunos em React

Este projeto foi desenvolvido como atividade da disciplina de **Programação Front-End**, utilizando **React + Vite**.

---

## 🚀 Tecnologias utilizadas

* React
* Vite
* JavaScript
* CSS

---

## 🎯 Objetivo

Criar um sistema de cartões de alunos utilizando componentes React, aplicando conceitos como:

* Componentes reutilizáveis
* Props (passagem de dados)
* Renderização dinâmica com `.map()`

---

## 💻 Funcionalidades

* Exibição de múltiplos alunos
* Cada aluno possui:

  * Nome
  * Turma
  * Cargo
  * Foto de perfil
  * Lista de habilidades
* Layout em formato de **cards**
* Tema **dark com destaque em roxo**
* Estilização com CSS
* Responsividade básica

---

## 🧩 Estrutura do projeto

```
src/
│
├── assets/            # Imagens dos alunos
├── components/        # Componentes reutilizáveis
│   ├── CartaoAluno.jsx
│   └── BadgeHabilidade.jsx
│
├── App.jsx            # Componente principal
├── App.css            # Estilos
├── main.jsx           # Renderização
```

---

## 🖼️ Preview

O projeto exibe cartões de alunos lado a lado, com foto, informações e habilidades em formato de badge.

---

## ⚙️ Como executar o projeto

1. Clonar o repositório: https://github.com/BaianoCoreano/cartao-alunos-react.git

```
git clone <https://github.com/BaianoCoreano/cartao-alunos-react.git>
```

2. Entrar na pasta:

```
cd meu-react
```

3. Instalar dependências:

```
npm install
```

4. Rodar o projeto:

```
npm run dev
```

5. Acessar no navegador:

```
http://localhost:5173
```
